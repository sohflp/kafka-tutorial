# Kafka Tutorial

The intent of this repository is to support local tests for beginners interested to explore Apache Kafka CLI.

## Commands

### Start Kafka cluster

```
make start
```

### Stop Kafka cluster

```
make stop
```

### Access Kafka shell for CLI commands

```
make shell
```
